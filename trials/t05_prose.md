Streamline 3.2 ships today with the features you asked for most. The new offline
workspace is available to all users at no cost — no plan changes, no upgrade
required. Drafts now sync across devices the moment you reconnect, and conflict
resolution shows a side-by-side diff instead of silently overwriting.

Under the hood, we rebuilt the sync fabric on three regional data centers, each
handling half of global traffic, which cut median sync latency from 900ms to
210ms in our load tests. Note that the offline workspace requires the Premium
tier and is enabled from Settings → Workspace → Offline.

Attachment previews now render for 40+ file types, and shared folders finally
support per-member permissions. As always, tell us what to build next in the
community forum — Streamline 3.2 is rolling out to everyone this week.

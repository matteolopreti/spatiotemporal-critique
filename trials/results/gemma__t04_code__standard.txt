**Smaller change that captures most of the gain:**
Change `clamp_all` to a generator and add type hints:

```python
from typing import Iterable, Iterator

def clamp(value: float, low: float, high: float) -> float:
    if low > high:
        raise ValueError(f"invalid range: low ({low}) > high ({high})")
    return max(low, min(value, high)) # Optimized for brevity/speed

def clamp_all(values: Iterable[float], low: float, high: float) -> Iterator[float]:
    """Clamp every value in `values` to [low, high]."""
    return (clamp(v, low, high) for v in values)
```

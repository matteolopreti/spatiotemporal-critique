Ledgerly 1.8 was released on March 12, 2025. This update focuses on quality-of-
life improvements requested by accounting teams.

CSV import now detects column headers automatically and previews the first ten
rows before committing, so mis-mapped columns can be corrected up front.
Recurring invoices support custom intervals from weekly to annually, and each
recurrence can carry its own note. Report exports include a new landscape PDF
layout that fits twelve months on one page.

Performance work in this release reduced report generation time by roughly 30%
on ledgers with more than 50,000 entries, measured on our standard benchmark
set. The mobile app receives the same importer improvements and adds fingerprint
unlock on supported devices.

Ledgerly 1.8 is a free update for all current subscribers and is available on
web, Windows, macOS, iOS, and Android today.
